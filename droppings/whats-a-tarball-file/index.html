What's a tarball ?

January 18, 2021 One-minute read
Linux
shell • tarball

A tarball, is a collection of files packed into a tar file for file storage. It’s often been compressed with gzip to save space. Thus the tar file has .tar or .tar.gz file extensions.

How Can We Unpack It ?

We can use tar command to unpack it in terminal?

# list the file in *tarball*
tar -tvf tarball
# extract the tarball
tar -xvf tarball
# create a tarball from a folder
tar -czvf tarball.tar.gz folder

Here are the options:

  • x: Extract file
  • v: Produce vervose mode
  • f: Specified the output filename
  • c: create a new tarball
  • t: list the file in the tarball
  • z: use z for .tar.gz, omits if have a .tar file

Install Apache Tomcat Server on Mac using Brew

January 18, 2021 2-minute read

Brew is a package manager tool for macOS, like apt for Ubuntu.

Install Tomcat Server

In this article, we just install apache tomcat 8.

# Install tomcat version 8
brew install tomcat@8

This will take care of the downloading, installation, configuration of tomcat and manager the dependencies as well. By default, Homebrew will install the package into the /usr/local/Cellar directory. You can check it:

# check the location
brew ls tomcat@8
# You get like these:

Directory Structure

List the content of /usr/local/Cellar/tomcat@8/8.5.61/libexec/

.
├── BUILDING.txt
├── CONTRIBUTING.md
├── bin
├── conf
├── lib
├── logs
├── temp
├── webapps
└── work
  • bin: contains binary files and scripts
  • conf: all configuration file can be find here
  • lib: contains packages which tomcat needed
  • logs: log file generated by tomcat in the process of running
  • temp: temporary file generated by tomcat in the process of runnning
  • webapps: contain all applications
  • work: contain compiled file, such as JSP compiled file.

The default port is 8080, you can edit /usr/local/Cellar/tomcat@8/8.5.61/libexec/conf/server.xml, change the port attribute.

<!-- inside server.xml -->
<Connector port="8080" protocol="HTTP/1.1"
               connectionTimeout="20000"
               redirectPort="8443" />

Run Tomcat Server

You can use brew services to manage all the services in you system.

# start the tomcat server
brew services start tomcat@8
# result
==> Successfully started `tomcat@8` (label: homebrew.mxcl.tomcat@8)

Once the server is runnning, you can visit the host page by http://localhost:8080.

Manage Applications

If you want to manage the applications runnig on the tomcat server, you should edit the configuration file: /usr/local/Cellar/tomcat@8/8.5.61/libexec/conf/tomcat-users.xml.

<!-- inside tomcat-users.xml -->
<tomcat-users>
...
<role rolename="manager-gui"/>
<user username="admin" password="se3ret" roles="manager-gui"/>
</tomcat-users>

You can set any username and password for your own. Now, restart the server.

brew services restart tomcat@8

You can get into the manager app page by click the Manager App on the host page after typing username and password.

Now you can view the application running the server, deploy your own application. The deloyed application can be located under the directory /usr/local/Cellar/tomcat@8/8.5.61/libexec/webapps/.
